## Step 3: Edit Colors (Optional)

Open `_variables.css` and change the colors:

```css
:root {
    --accent-blue: #FF6B6B;     /* Your primary color */
    --bg-body: #FFF5E6;          /* Background color */
    --text-primary: #2D3436;     /* Main text color */
}
```

## Quick Color Palette

| Variable | Purpose |
|----------|---------|
| `--accent-blue` | Primary buttons & accents |
| `--accent-green` | Success states |
| `--accent-red` | Danger/delete buttons |
| `--bg-body` | Page background |
| `--text-primary` | Main text color |
| `--text-secondary` | Muted text color |
