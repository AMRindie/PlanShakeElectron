# PlanShake

Electron based web app for project management, runs fully offline.

## Features
- Persistent Local data storage (IndexedDB)

## How To install

Download the latest release from the [releases page](https://github.com/AMRindie/PlanShakeElectron/releases) and run the installer.

## Auto‑Update
The app checks for updates on launch (after 3 s) and hourly thereafter. When a new release is published on GitHub, users receive a notification with a one‑click download and install option.
